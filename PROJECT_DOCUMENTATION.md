# 📱 ChatChain - Полное описание проекта

## 🎯 Концепция
**ChatChain** - современный чат-мессенджер в стиле Discord с поддержкой комнат, голосовых сообщений, музыки, игр и системы геймификации.

---

## 🏗️ Архитектура

### Технологический стек:
- **Frontend**: Next.js 15 (App Router), React 18, TypeScript
- **Стили**: TailwindCSS + кастомные CSS переменные
- **Иконки**: Lucide React
- **UI компоненты**: shadcn/ui
- **Бэкенд**: Next.js API Routes
- **Хранилище**: localStorage + in-memory storage

### Структура проекта:
```
/src
├── app/
│   ├── page.tsx           # Главная страница чата
│   ├── layout.tsx         # Корневой layout
│   ├── globals.css        # Глобальные стили (9900+ строк)
│   └── api/               # API endpoints
│       ├── chat/          # Чат API
│       ├── auth/          # Авторизация
│       ├── music/         # Музыка
│       ├── moderation/    # Модерация
│       ├── leaderboard/   # Рейтинг
│       ├── gif/           # GIF пикер
│       └── gif-proxy/     # Прокси для изображений
├── components/
│   ├── Header.tsx         # Шапка сайта
│   ├── RoomsSidebar.tsx   # Сайдбар комнат
│   ├── GifPicker.tsx      # Выбор GIF
│   ├── EmojiPicker.tsx    # Выбор эмодзи
│   ├── SharedMusicPlayer.tsx # Музыкальный плеер
│   ├── AdminPanel.tsx     # Панель администратора
│   ├── AuthModal.tsx      # Модалка авторизации
│   └── ...                # Другие компоненты
└── lib/
    ├── store.ts           # Состояние и типы данных
    ├── translations.ts    # Мультиязычность (RU/EN)
    ├── db.ts              # База данных
    └── ...                # Утилиты
```

## 📊 API ENDPOINTS

### `/api/chat`
- `GET` - Получить сообщения комнаты
- `POST` - Отправить сообщение, создать комнату, и т.д.

### `/api/auth`
- Регистрация и авторизация пользователей

### `/api/music`
- Управление музыкальным плеером

### `/api/leaderboard`
- Топ пользователей по сообщениям

### `/api/moderation`
- Модерация пользователей

### `/api/gif`
- Получение GIF по категориям

### `/api/gif-proxy`
- Прокси для обхода CORS

## 🎨 Компоненты

### Основные:
- `Header` - Шапка с навигацией, темой, уведомлениями
- `RoomsSidebar` - Список комнат
- `SharedMusicPlayer` - Музыкальный плеер
- `AdminPanel` - Панель администратора
- `AuthModal` - Модальное окно авторизации

### Пикеры:
- `GifPicker` - Выбор GIF (8 категорий, 96+ GIF)
- `EmojiPicker` - Выбор эмодзи
- `StickerPicker` - Выбор стикеров

### Модальные окна:
- `CreateRoomModal` - Создание комнаты
- `PasswordModal` - Ввод пароля комнаты
- `InviteModal` - Приглашение пользователей

### Эффекты:
- `HeartsEffect` - Анимация сердечек
- `Confetti` - Анимация конфетти при триггерах
