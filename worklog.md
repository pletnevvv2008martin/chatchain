## Структура проекта ChatChain

### API маршруты:

## Игра Fortress - Текущее состояние

### Реализовано:

#### 1. Система рас (10 рас):
| Раса | Эмодзи | Особенность |
|------|--------|-------------|
| Люди | 👤 | +20% экономика, баланс |
| Эльфы | 🧝 | +50% магия |
| Гномы | 🪓 | +50% защита |
| Тёмные эльфы | 🦇 | +40% атака ночью |
| Нежить | 🦴 | Воскрешение павших |
| Оборотни | 🐺 | +100% в полнолуние |
| Орки | 🧌 | +30% размер армии |
| Маги | 🧙 | x2 магия |
| Вампиры | 🧛 | Вампиризм |
| Дракониды | 🐲 | Летающие юниты |

#### 2. Здания (20 типов):
- **Экономические:** Замок, Ферма, Шахта, Лесопилка, Склад, Рынок, Мастерская, Сокровищница
- **Военные:** Казарма, Стена, Башня, Арена, Кузница, Конюшня
- **Магические:** Храм, Башня мага, Библиотека, Обсерватория
- **Специальные:** Часовня (свадьбы), Таверна (герои)

#### 3. Ресурсы (6 типов):
- 💰 Золото
- 🪨 Камень
- 🪵 Дерево
- ⚙️ Железо
- 🍖 Еда
- 🔮 Мана

#### 4. Карта мира:
- Гексагональная сетка (30x20)
- Типы terrain: трава, лес, гора, вода, пустыня, снег, болото, лава
- Система тумана войны
- Мини-карта

#### 5. Объекты на карте:
- Ресурсные точки (золотые шахты, каменоломни, фермы и т.д.)
- Монстры (гоблины, орки, драконы, некроманты, демоны, титаны)
- Замки игроков

#### 6. Механики:
- Движение героя по карте
- Атака монстров
- Захват ресурсных точек
- Пошаговая система (очки движения)
- Пассивный доход ресурсов
- Система боя с расчётом потерь

#### 7. Интерфейс:
- Выбор расы при первом входе
- Вкладки: Карта, Замок, Статистика
- Модальные окна боя и захвата
- Масштабирование и перемещение карты
- Фоновая музыка

### Файловая структура Fortress:
```
/src/app/fortress/page.tsx    - Основная страница игры
/src/lib/fortress.ts           - Логика игры (расы, здания, ресурсы)
/src/app/api/fortress/route.ts - API для сохранения данных
/data/fortress/{userId}.json   - Данные крепостей пользователей
```

### API Fortress:
- `GET ?userId=xxx` - Получить данные крепости
- `GET ?action=leaderboard` - Рейтинг игроков
- `POST` - Создать крепость (выбор расы)
- `PUT action=upgrade_building` - Улучшить здание
- `PUT action=train_units` - Обучить юнитов
- `PUT action=collect_resources` - Собрать ресурсы
