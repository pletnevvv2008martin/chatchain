# Настройка GIF

## Вариант 1: GIPHY API (рекомендуется)

1. Зарегистрируйтесь на https://developers.giphy.com/
2. Создайте приложение и получите API ключ
3. Добавьте ключ в `.env.local`:

```
GIPHY_API_KEY=ваш_ключ_здесь
```

После этого GIF будут загружаться напрямую из GIPHY с поиском.

## Вариант 2: Локальные GIF

1. Скачайте GIF файлы
2. Положите их в папки:
   - `/public/gifs/trending/`
   - `/public/gifs/reactions/`
   - `/public/gifs/funny/`
   - `/public/gifs/love/`
   - `/public/gifs/celebrate/`
   - `/public/gifs/animals/`
   - `/public/gifs/cool/`
   - `/public/gifs/memes/`

## Вариант 3: Статические GIF (по умолчанию)

Работает автоматически - использует встроенную коллекцию из 96 GIF.
Если внешние ссылки не грузятся - автоматически используется прокси.

## Кнопки в интерфейсе

- 🔄 **Обновить** - перезагрузить GIF текущей категории
- 🔗 **Открыть GIPHY** - перейти на giphy.com для поиска GIF
- **Поле ввода** - вставить ссылку на любой GIF
